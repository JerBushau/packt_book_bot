FreebookApp

Never miss another Packt Publishing free learning offer again!

FreebookApp is a slack application that provides one slash command with a few options. Created to help keep you and your team up to date on the awesome Packt Publishing free learning offer.

Everything you need to know:

/freebook

Sends user a private message containing the free learning offer and link to log in and download.

/freebook public

Sends current channel a message containing the free learning offer and link to log in and download.

/freebook x

Schedules a reminder to post the free learning offer in the current channel, daily at the specified time. Where x is a number 0-23. To set a reminder for 8am daily: /freebook 8, for 2pm daily: /freebook 14 etc..

/freebook cancel

Cancels the teams scheduled reminder.

/freebook help

Sends user a message with helpful information about FreebookBot
